[HC-05]:
接线:
	VCC<---->3V3
	GND<---->GND
	TX<---->P30
	RX<---->P31
	
	单片机的P30是RX
注意事项:
	1.蓝牙地址:20:19:11:15:36:16
	2.蓝牙ID:
	3.任意蓝牙连接模式
	4.PC直接连接该蓝牙接口,选择无配对模式.第一次连接需要等待串口驱动的安装.(时间较长)
