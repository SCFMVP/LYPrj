[DHT11]:
接线:
	VCC<---->3V3
	GND<---->GND
	DATA<---->P20
注意事项:
	1. DATA为温湿度单工数据线,在DHT11.h中修改引脚
	
[HC-05]:
接线:
	VCC<---->3V3
	GND<---->GND
	TX<---->P30
	RX<---->P31
注意事项:
	1.蓝牙地址:20:19:11:15:36:16
	2.蓝牙ID:
	3.任意蓝牙连接模式
	4.PC直接连接该蓝牙接口,选择无配对模式.第一次连接需要等待串口驱动的安装.(时间较长)

三线光敏:
接线:
	VCC<---->3V3
	GND<---->GND
	Do<---->P32	
注意事项:
	1.三线光敏具有方向性，只感应传感器正前方的光源，用于寻光效果更佳
	2.无模拟输出,只输出0和1,有光输出0,无光输出1
